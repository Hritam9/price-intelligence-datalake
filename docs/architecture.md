# Architecture – Competitive Price Intelligence Data Lake

1️⃣ **Extraction:** Collect product price data from Amazon, Flipkart, Myntra.  
2️⃣ **Transformation:** Merge and clean datasets using Pandas.  
3️⃣ **Load:** Store processed results into Azure Data Lake.  
4️⃣ **Analytics:** Run price recommendation logic.  
5️⃣ **Visualization:** Streamlit dashboard to view insights.
